# Azure Machine Learning Service 动手实践

Azure 机器学习服务是一项云服务，可以使用它来训练、部署、自动执行以及管理机器学习模型，并提供训练和部署的灵活性。本模块介绍如何使用 AML Service 实现自动化机器学习，超参数优化，云端部署及本地部署等。
